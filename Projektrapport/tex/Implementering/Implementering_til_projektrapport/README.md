# LaTeXSkabelon
LaTeX skabelon tom for alt muligt andet end selve skabelonen.

# Brug af skabelon
Brug den ved at pull, lav nyt repo, kopier det ind i, og arbejd der, i må ikke pushe til dette repo!

I skal kun ændre i config og main filerne ingen af de andre filer skal i røre, da dette gør at jeg kan updatere skabelonen 
uden at i skal ændre en masse i jeres repos.
